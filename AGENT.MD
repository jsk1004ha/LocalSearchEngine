# agent.md

## 목적
이 저장소는 "웹 전용 고성능 조사형 검색 엔진"의 구현/개선을 목표로 합니다.

## 작업 원칙
1. 검색 결과는 `결론 + 근거 + 반례 + 진단`을 함께 제공한다.
2. 모든 검색은 온라인 웹 검색 기반으로 수행한다.
3. 온라인 질의는 시간 예산을 준수하고, 생략된 단계가 있으면 diagnostics에 기록한다.
4. 근거 출처(`SourceCitation`)를 유지하고 추적 가능해야 한다.

## 코드 규칙
- Python 3.11+ 기준 작성.
- 새 기능은 가급적 `schema -> retrieval -> engine` 경계를 유지한다.
- 하드코딩된 임계값 추가 시 주석으로 의도를 남긴다.

## 검증 규칙
- 최소한 `python -m compileall src` 성공 확인.
- 핵심 변경 시 간단한 실행 예제로 `SearchResult` 필드 일관성 확인.

## 문서 규칙
- 사용자 가이드는 `README.md`에 유지.
- 버전 변경은 `VERSION` 파일을 단일 소스로 갱신.
