# Code of Conduct

이 프로젝트는 모든 기여자가 안전하고 생산적으로 협업할 수 있는 환경을 지향합니다.

## 기대 행동

- 상호 존중을 지키고, 공격적/차별적 발언을 하지 않습니다.
- 기술적 논쟁은 근거와 재현 가능한 정보에 기반해 진행합니다.
- 초보자/외부 기여자에게 친절한 커뮤니케이션을 권장합니다.

## 허용되지 않는 행동

- 괴롭힘, 혐오 표현, 위협, 성적/폭력적 콘텐츠
- doxxing(개인정보 노출) 또는 동의 없는 개인정보 공유
- 반복적 트롤링/스팸

## 신고

공개적으로 민감한 내용이 발생했다면, 가능하면 공개 이슈 대신 **비공개 채널**을 우선 사용해 주세요.
비공개 신고 채널이 없다면, 공개 이슈에는 최소한의 정보만 남기고 추가 조율을 요청해 주세요.
